# Movie UI Practice (HTML + Tailwind CSS)

## Overview

This project is a small practice website for a movie platform interface and a simple sign‑in page.  
It is built using **only HTML and Tailwind CSS**, with **no JavaScript**.

The focus of the project is:
- **Clean, modern UI**
- **Responsive layouts** that work on mobile, tablet, and desktop
- Simple, accessible components for learning and practicing frontend structure

## Pages

- **`agasbnytailwin.html`** – main movie home page with “New Releases” cards.
- **`series.html`** – TV series listing page using the same navigation and style.
- **`categories.html`** – categories overview (Action, Comedy, Horror, Drama, etc.).
- **`index.html`** – sign‑in page for users to log in.

All pages share a consistent navigation and visual style.

## Technologies

- **HTML5** – page structure and content.
- **Tailwind CSS (via CDN)** – utility‑first styling and responsive design.
  - Included using a pure CSS `<link>` tag (no Tailwind JavaScript).
